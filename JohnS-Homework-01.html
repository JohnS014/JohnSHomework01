Github Workflow

a. What is a commit?

a commit is where you add your code to a branch

b. What is a branch?

a branch is a seperate working area for code under development

c. What is a pull request?

a pull requets is when you want to merge your code from a local repository or 'feature branch' back into the master repository.

d. Other information beginner coders would find useful.

you can download the github desktop app, sign up to the website or use the terminal/bash on your machine to create repositories, fetch the latest changes to code and merge your own commits back into the master repository.

A description of the Unix Shell

a. How to list the files in a directory.

in the terminal enter ls -F

b. How to show and change the working directory.

in the terminal enter pwd to print the working directory you are in and enter cd foldername to change the directory

c. How to create a new directory.

enter mkdir name in the terminal to create a new directory

d. How to create and remove files.

enter touch name.type in the terminal to create a new file, to remove files enter rm name.type
